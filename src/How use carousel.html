Carousel function

CreateCarousel()

Podemos criar um carousel de loader que posteriormente será substituido por um com os valores necessários.

Parâmetro Função
idAppend É o ID da onde o carousel fará o append. default = no body do document
array Array no qual irá carregar os valores. Default=loader
titleParameter O título do carousel. Default="Loader"
badgeStyle Estilo do badge. Opções: Triangle ou default (sem parâmetro)

Abaixo temos um carousel "Loader" passando no primeiro parâmetro com a string "loader."

createCarousel("carousel");
